# market-terminal — Live Market Dashboard

## Project
Real-time stock market dashboard: Watchlist, Portfolio P&L, Alerts, News.

## Structure
- `index.html` — main dashboard
- `style.css` — dashboard styles
- `main.js` — dashboard logic (Yahoo Finance API, charts)

## Deploy
GitHub Pages from `main` branch root.

## Branch strategy
- `main` — production only, via PR
- `dev` — integration branch
- `feat/*` — new features
- `fix/*` — bugfixes
